1. Bootstrap 5 มีอะไรใหม่ๆบ้างที่แตกต่างจาก Version เก่า อย่างน้อย 5 ตัวอย่าง

1. เพิ่มการซัพพอร์ตเว็บแบบขวาไปซ้าย

2. ปรับปรุงและเพิ่มคอมโพเนนต์ "forms"

3. หยุด Support เบราว์เซอร์บางตัว เช่น Internet Explorer, Microsoft Edge Legacy เป็นต้น

4. คลาสใหม่ "placeholder"

5. คอมโพเนนต์ใหม่ "offcanvas"

2. จุดเด่นของ Bootstrap คืออะไร จงอธิบาย

Responsive Design ที่พร้อมใช้งาน:

- Bootstrap มีระบบ Grid ที่สามารถปรับแต่งและใช้ง่าย ช่วยให้เว็บแสดงผลที่เหมาะสมกับหลากหลายอุปกรณ์

3. จงอธิบายการติดตั้งและใช้งานแบบ CDN และแบบ Internal (Compiled CSS and JS) แตกต่างกันอย่างไร

แบบ CDN:

- สามารถเพิ่มลิงค์ไปยังไฟล์ CSS และ JavaScript ของ Bootstrap จาก CDN ลงในไฟล์ HTML ทำให้เว็บสามารถใช้งาน Bootstrap ได้เลย

แบบ Internal:

- สามารถใช้แบบ Offline ได้ โดยไม่จำเป็นต้องเชื่อมต่ออินเตอร์เน็ต โดยดาวน์โหลดจากในเว็บ Bootstrap
